# Nosok Bot (Trial Version)

A powerful Discord bot with moderation, fun, utility, and voice commands.

## Features
- 🛡️ Moderation (kick, ban, timeout, warn, mute, clear)
- 🎉 Fun commands (roll, coinflip, joke, 8ball, rps)
- 🛠️ Utility (poll, timer, AFK, calculator, server icon/banner)
- 🔊 Voice (temporary VC, move members, voice info)
- 📊 Warning system with SQLite database
- 🔒 Configurable log channel
- ✨ Beautiful embeds with custom logo and banner

## Installation
1. Install Python 3.8+
2. Install dependencies: `pip install -r requirements.txt`
3. Configure `config/config.cfg` with your bot token
4. Place your `banner.jpg` and `logo.jpg` in the `res/` folder
5. Run: `python main.py`

## Commands
Use `/help` to see all commands.
